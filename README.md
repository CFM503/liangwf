# LiangWF — 个人量化交易项目

> 从校园宿舍开始的量化之路

包含两个子项目：

| 目录 | 说明 |
|------|------|
| [`XiaoLiangTrader/`](XiaoLiangTrader/) | **完整量化系统** — 策略 + ML + 自动交易 + 风控 |
| 根目录文件 | 早期回测脚本（Backtrader / VectorBT） |

## 快速开始

```bash
# 进入主项目
cd XiaoLiangTrader

# 安装依赖
pip install -r requirements.txt

# 回测
python main.py --backtest

# 详细文档见 XiaoLiangTrader/README.md
```
