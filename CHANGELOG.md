## v1.0.3（2026-06-28）

### 🔍 新增：全市场扫描 — 从 5000+ 只 A 股中选股

**一句话：** 不再局限于固定的 20 只股票，现在能扫描整个 A 股市场（5800+ 只），自动筛选出最值得买入的 10 只。

**新增文件：**
- `data/fetcher.py` 新增 `fetch_all_a_snapshot()` — 获取全 A 股实时快照
- `strategy/screener.py` 新增 `scan_all()` — 两阶段全市场扫描

**新增命令：**
```bash
# 全市场扫描（从 5000+ 只 A 股中选）
python main.py --scan --all

# 扫描指定股票池（原有功能不变）
python main.py --scan

# 指定几只股票
python main.py --scan --pool 600519 300750 002594
```

**两阶段扫描原理：**

| 阶段 | 做什么 | 耗时 |
|------|--------|------|
| 第一阶段 | 获取全 A 股实时快照，用**量比、涨跌幅、换手率、市值**快速筛选 | ~2 秒 |
| 第二阶段 | 拉取候选股票的历史数据，做完整技术面打分（均线、MACD、RSI 等） | ~2 分钟 |

**第一阶段快速筛选条件：**
1. 排除 ST、退市、新股（名称含 ST / N / C）
2. 排除价格 < 2 元 或 > 500 元
3. 排除停牌（成交额为 0）
4. 只保留沪深主板 + 创业板（代码以 0/3/6 开头）
5. 按换手率、涨跌幅、市值综合打分，取前 200 名

**一句话：** 不用一只一只看股票了，程序帮你扫描整个市场，自动排名，告诉你今天该买哪只。

**打分逻辑（5 个维度，每个 0~100 分）：**

| 维度 | 权重 | 什么情况得高分 | 什么情况得低分 |
|------|------|----------------|----------------|
| **均线** | 30% | 金叉刚发生（+20）、多头排列（+20） | 死叉（-30）、价格在均线下（-10） |
| **成交量** | 20% | 温和放量上涨（+30）、连续3天放量（+10） | 天量（-15，可能见顶）、缩量下跌（-20） |
| **MACD** | 20% | 金叉（+25）、红柱放大（+15） | 死叉（-25） |
| **RSI** | 15% | 30~50 超卖回升（80分，最佳买点） | >80 超买（20分，危险） |
| **趋势** | 15% | 20日斜率为正（+25）、接近新高（+10） | 斜率为负（-25）、接近新低（-10） |

**如果开了 ML 模型：** 技术面 40% + ML 预测概率 60% = 最终得分

**仓位分配：** 得分高的股票分到更多钱，但单股不超过 20%，总仓位不超过 60%。

```
├── bot/                 ← 自动交易（执行器 + 风控 + 通知）
└── utils/               ← 工具（日志 + 加密）
```

### 📈 核心策略：双均线交叉 + 成交量过滤

**新增文件：**
- `strategy/signals.py` — 信号定义（BUY/SELL/HOLD）
- `strategy/dual_ma.py` — 双均线策略

**买入条件（同时满足）：**
1. MA5 上穿 MA20（金叉）— 短期趋势向上
2. 当日成交量 > 20日均量 × 1.5 — 有人在买

**卖出条件（任一触发）：**
1. MA5 下穿 MA20（死叉）— 趋势反转
2. 亏损达 -8%（止损）— 认输离场
3. 盈利达 +15% 后从最高点回撤 5%（跟踪止盈）— 锁定利润

### 🤖 ML 增强：用机器学习过滤假信号

**新增文件：**
- `strategy/ml_strategy.py` — ML 增强策略
- `ml_model/features.py` — 30 个技术指标特征
- `ml_model/predictor.py` — 统一预测器（支持 4 种模型）

**为什么加 ML？** 双均线策略在震荡市会频繁"假金叉"→ 买入 → 止损，亏钱。
ML 的作用是：只在模型认为"大概率涨"时才买入，过滤掉假信号。

**支持 4 种模型（config.yaml 中 `ml.model_type`）：**

| 模型 | 说明 | 需要安装 |
|------|------|----------|
| `lightgbm` | 速度最快，推荐 | `pip install lightgbm` |
| `xgboost` | 精度略高 | `pip install xgboost` |
| `random_forest` | 最稳定，不过拟合 | sklearn 自带 |
| `gradient_boosting` | sklearn 自带 GBDT | sklearn 自带 |

**30 个特征：**
- 均线偏离率（价格离均线多远）
- 1/5/10/20 日收益率（涨了多少）
- 波动率、ATR（波动大不大）
- 量比、量价相关性（有没有人买卖）
- RSI、MACD、KDJ、布林带（经典技术指标）
- K线形态（上下影线、实体大小）
- 趋势斜率（整体向上还是向下）

### 🤖 自动化交易

**新增文件：**
- `bot/executor.py` — 模拟盘执行器（SQLite 持久化）
- `bot/risk.py` — 风控（Kill Switch + 涨跌停 + 仓位）
- `bot/notifier.py` — 邮件通知
- `bot/scheduler.py` — 每日自动编排

**模拟盘特性：**
- 佣金万2.5 + 印花税千一 + 0.1% 滑点
- A股100股整数
- T+1限制（今天买的今天不能卖）
- 全部存 SQLite，重启不丢状态

**风控机制：**
- **Kill Switch**：创建 `.kill_switch` 文件或运行 `--stop` 立即停止
- **涨跌停**：涨停不追（≥9.5%），跌停不卖（≤-9.5%）
- **止损**：单笔 -8% 强制卖出
- **跟踪止盈**：盈利 15% 后从最高点回撤 5% 卖出
- **仓位限制**：单股 ≤ 20%，总仓位 ≤ 60%
- **日亏损限额**：当日 ≥ 3% 暂停交易
